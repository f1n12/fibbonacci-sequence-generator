# Fibonacci Sequence Generator
 
A command-line program built in C++ that generates the nth number in the Fibonacci sequence.
 
## Features
 
- Calculates the nth Fibonacci number based on user input
- Iterative implementation for efficiency
- Handles invalid input (0)
## How to run
 
Make sure you have a C++ compiler installed (g++ recommended).
 
```
g++ fibonacci_sequence.cpp -o fibonacci_sequence
./fibonacci_sequence
```
